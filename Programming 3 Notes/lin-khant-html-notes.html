HTML Table Tutorial

Tags Definition
<table>…</table> Mark the beginning of the table
<tr>…</tr> Mark the beginning and end of one table row
<th>…</th> Mark the beginning and end of one table header cell.
<td>…</td> Mark the beginning and end of one table data cell.
border="1" You can put this in the <table> tag to show borders

Table Example

<table border="1">
 <tr>
  <th>Head 1</th><th>Head 2</th><th>Head 3</th>
 </tr>
 <tr>
  <td>R1C1</td><td>R1C2</td><td>R1C3</td>
 </tr>
 <tr>
  <td>R2C1</td><td>R2C2</td><td>R2C3</td>
 </tr>
 <tr>
  <td>R3C1</td><td>R3C2</td><td>R3C3</td>
 </tr>
</table>
Head 1 Head 2 Head 3
R1C1 R1C2 R1C3
R2C1 R2C2 R2C3
R3C1 R3C2 R3C3
